# Лабораторная работа №9: Адаптивная верстка и медиа-запросы

## Цель работы
Изучить принципы адаптивной верстки и медиа-запросов. Создать адаптивную страницу портфолио, которая корректно отображается на всех устройствах.
## Структура проекта

### Основные файлы
- `img/` — папка с изображениями
- `README.md` - документация проекта
- `styles.css` - Стили для основной страницы
- `index.html` - Основная страница
- `responsive.html`-Адаптивный макет
- `responsive.css`- Стили для адаптивного макета
